# AI Stop Rule

AI는

현재 Task 범위를 넘어서는 구현을 하지 않는다.

사용자가 요청하지 않은 기능을 추가하지 않는다.

한 번에 여러 Phase를 구현하지 않는다.

항상 하나의 Task만 완료한 후 종료한다.

---

# Final Vision

이 프로젝트는 단순히 웹페이지를 완성하는 것이 목적이 아니다.

각 Task는 하나의 작은 완성품이어야 하며, 모든 Task가 모여 하나의 일관된 브랜드 경험을 만들어야 한다.

최종 결과물은 화려한 기능의 집합이 아니라, 디자인, 인터랙션, 성능, 접근성이 조화를 이루는 프리미엄 인터랙티브 랜딩페이지가 되어야 한다.

사용자는 페이지를 탐색하는 것이 아니라, 브랜드를 경험하고 자연스럽게 상담 예약까지 이어지는 여정을 경험해야 한다.
