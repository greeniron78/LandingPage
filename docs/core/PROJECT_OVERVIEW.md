# Project Vision

이 프로젝트는 일반적인 피부샵 홈페이지를 만드는 것이 아니다.

광고(네이버, 인스타그램, 카카오, 구글 등)를 통해 유입된 고객이 페이지에 들어오는 순간부터 예약을 완료하기까지 하나의 프리미엄 브랜드 경험을 제공하는 인터랙티브 랜딩페이지를 제작한다.

방문자는 단순히 정보를 읽는 것이 아니라, 스크롤을 내릴수록 피부가 점차 건강하게 변화하는 경험을 통해 브랜드의 전문성과 신뢰를 자연스럽게 느끼고 상담 예약까지 이어져야 한다.

최종 목표는 아름다운 홈페이지가 아니라 높은 예약 전환율(CVR)을 만드는 것이다.

---

# Core Goal

이 랜딩페이지의 목적은 단 하나이다.

"광고 유입 사용자를 상담 예약으로 전환한다."

모든 디자인과 인터랙션은 이 목표를 위해 존재한다.

불필요한 메뉴나 복잡한 정보 구조는 최소화한다.

# UI Principles

여백을 충분히 사용한다.

복잡한 장식은 사용하지 않는다.

강한 컬러 사용을 최소화한다.

화이트 기반 디자인을 사용한다.

골드 포인트 컬러는 강조 요소에만 사용한다.

카드는 둥근 모서리를 사용한다.

그림자는 매우 은은하게 사용한다.

---

# Motion Principles

# Success Criteria

이 프로젝트가 성공했다고 판단하는 기준은 다음과 같다.

- Hero 인터랙션이 자연스럽고 인상적이다.
- 모바일에서 부드럽게 동작한다.
- 사용자가 끝까지 스크롤하고 싶어진다.
- 브랜드가 프리미엄으로 인식된다.
- 예약 버튼이 자연스럽게 클릭된다.
- 정보 전달보다 경험 중심의 랜딩페이지가 완성된다.

---

# Final Vision

이 랜딩페이지는 단순히 피부샵을 소개하는 웹사이트가 아니다.

광고를 통해 처음 방문한 사용자가 브랜드를 신뢰하고, 피부가 변화하는 경험을 간접적으로 체험하며, 상담 예약까지 이어지는 하나의 디지털 경험을 제공해야 한다.

최종 결과물은 국내 최고 수준의 프리미엄 뷰티 브랜드 랜딩페이지를 목표로 하며, Apple의 부드러운 인터랙션, Aesop의 미니멀한 감성, Sulwhasoo의 프리미엄 브랜드 경험을 참고하되 독창적인 사용자 경험으로 완성한다.
