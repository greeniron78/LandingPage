# Premium Skin Care Interactive Landing Page

> **A premium interactive beauty landing page built with Next.js, TypeScript, GSAP, Framer Motion, and Tailwind CSS.**

---

## ✨ Overview

Premium Skin Care Interactive Landing Page는 일반적인 피부샵 홈페이지가 아닙니다.

광고를 통해 유입된 사용자가 브랜드를 경험하고, 자연스럽게 상담 예약까지 이어질 수 있도록 설계된 **프리미엄 인터랙티브 랜딩페이지**입니다.

이 프로젝트는 다음 네 가지를 가장 중요한 가치로 생각합니다.

- Premium Brand Experience
- Interactive Storytelling
- High Performance
- High Conversion
